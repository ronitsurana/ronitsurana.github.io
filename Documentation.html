ClockCraft Documentation

A modern, customizable multi-clock application built with vanilla JavaScript and Canvas.

Overview

ClockCraft lets you create multiple analog clocks on a single page, each displaying a different time zone. The application features a dark glassmorphism UI, premium clock rendering, and built-in timer/stopwatch tools.

Getting Started

Opening the App

Open ClockCraft.html in a modern browser. Four default clocks will appear (Asia/Kolkata, America/New_York, Europe/London, Asia/Tokyo).

Adding Clocks

Click the "+ Add Clock" button in the header. You can add as many clocks as you need.

Choosing Clock Size

Use the size dropdown in the header to switch between Small (220px), Medium (280px), and Large (360px). All clocks resize together.

Clock Customization

Click any clock face or the "⚙ Settings" button to open the settings sidebar.

Hand Color

Use the color picker to change the hour and minute hand color. The second hand always uses the accent orange.

Time Zone

Select any IANA time zone from the dropdown. The clock and its label update immediately on Apply.

Background Image

  • Click "Choose an image…" to select a local image file.
  • On Apply, the image is set as the clock's background and you're prompted to auto-set the hand color for contrast.
  • Click "Remove BG" in the sidebar to clear the background.

Timer & Stopwatch

Timer

Enter hours, minutes, and seconds in the sidebar "Timer" section and click Start. A modal notification appears when time is up.

Quick Timer

Enter a number of seconds in the "Quick Timer" field and click Start. Great for short countdowns.

Stopwatch

Click ▶ Start to begin. The display updates live. Click ⏸ Pause to pause, and ↺ Reset to clear.

Picture-in-Picture (PiP)

With a clock selected in the sidebar, click "PiP" to pop the clock into a floating browser window. Closing the PiP window returns the clock to the page. Requires Chrome 116+ with Document PiP support.

Architecture

ClockCraft Class

class ClockCraft {
  constructor(container, { width, height, color, timezone })
  setColor(hex)           // Change hand color
  setTimezone(tz)         // Change IANA timezone
  setBackgroundImage(url) // Set background image
  removeBackgroundImage() // Clear background
  getComplementColor()    // Get complement of avg background color
  resize(size)            // Resize the clock
  openConfig()            // Open sidebar for this clock
  destroy()               // Remove clock from page
  static clocks           // Array of all clock instances
  static activeClock      // Currently selected clock
}

Rendering

Each clock uses a single <canvas> element rendered at 60fps via requestAnimationFrame. The clock face includes: radial gradient background, minor & major tick marks, hour numbers, tapered hour/minute hands, a glowing second hand with tail, and a gradient center dot.

Dependencies

  • Bootstrap 5.3.8 — loaded via jsDelivr CDN (CSS + JS bundle)
  • Inter font — loaded via Google Fonts
  • Removed Colors.js — replaced with a native 3-line complement color function

Browser Support

  • Chrome 90+ (PiP requires 116+)
  • Firefox 90+
  • Safari 15+
  • Edge 90+
